# phase-4-data-structures

This repository documents **Phase 4** of my structured Software Engineering learning journey, focusing on one of the most fundamental topics in computer science: **Data Structures**.

The goal of this phase is to understand how data is organized, stored, accessed, and manipulated efficiently. Choosing the right data structure is essential for writing fast, scalable, and maintainable software.

This phase builds upon computational thinking and problem-solving skills from previous phases and prepares the foundation for algorithms, databases, system design, and technical interviews.

---

## Topics Covered

- Arrays
- Strings
- Linked Lists
- Stacks
- Queues
- Hash Tables
- Trees
- Graphs
- Recursion
- Memory Usage of Data Structures

---

## Skills Gained

After completing this phase, I will understand:

- How data is stored in memory
- Choosing the right data structure for different problems
- Advantages and disadvantages of common data structures
- Time and space efficiency
- Linear and non-linear data structures
- Relationships between different data structures
- How modern applications organize data
- Foundations for learning algorithms
